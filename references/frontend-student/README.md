# Student Portal

React + Vite frontend for the BITS student portal (separate domain from the admin/staff portal).

## Development

```bash
npm install
npm run dev
```

Runs at http://localhost:5174 and proxies `/api` to the backend at http://localhost:5001.

## Related

- Admin/staff portal: `../frontend-admin` (http://localhost:5173)
- Backend API: `../backend` (http://localhost:5000/api/v1)
